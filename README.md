# Imagick

Remove Image Background using PHP imagick

## Requirements 

PHP imagick extension

## Install extenstion

Linux/Debian

```bash
sudo apt-get install php-imagick
#Verify if imagick has been loaded as an extension (optional)
php -m | grep imagick
#Restart Apache
sudo service apache2 restart
```
